# A Ray Tracing Engine

See the examples directory for some cool renders using this engine!

## How to run this engine
```
git clone https://github.com/mxmlkzdh/RayTracing.git
cd RayTracing
mkdir build
cd build
cmake ..
cmake --build .
cd ..
./build/main
```
